A guide to how much RAM your Apple computer has installed

  • To identify how much memory your Mac currently has installed, go to the Apple menu in the upper left-hand corner of your screen. Select "About This Mac" from the menu. That screen will show the total amount of memory (RAM) your Apple has installed.
  • Next click on the "More Info" button on the "About This Mac" screen. The System Profiler will appear. Select "Memory" from the options on the left column. This screen will show how many memory slots - sockets you have in your system to install memory (chips) and what they currently have installed. Keep in mind that if your slots are full, you will need to remove existing RAM to install larger capacity memory modules.
